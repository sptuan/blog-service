# DangoBlog-Service
## 简述
DangoBlog🍡，中文名团子博客🍡，目前定位为个人练手项目。
旨在熟悉Go语言，并熟悉常用的web应用开发流程，并享受数据流动的乐趣。
## 期望功能
- 文章和附件发布修改
- 多媒体附件管理
- 搜索和标签系统
- ...
